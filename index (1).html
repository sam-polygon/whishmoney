    How it works behind the scenes

    One single new piece. The rest is already running.

    Whish provides neobank experiences to Lebanese and MENA users. We are now providing USD access via stablecoins. A single API surface - Polygon OMS - handles every flow: wallets, ramps, cross-chain, and payouts. Polygon Trails routes any stable from any chain to Polygon. The 1:1 WUSD Stable Swap is the only new piece - permissioned for authorized minters. Anchorage handles issuance and Polygon handles self-custodial wallets through existing regulated infrastructure.

    Live today

    Trails Intent Address

    The smart contract that accepts any stable from any network and resolves the optimal route to Polygon. Already powering live cross-chain swaps across EVM, Solana, and Tron.

    New for Whish

    1:1 WUSD Stable Swap

    A permissioned pool on Polygon that swaps any liquid stable (USDC, USDT) for WUSD at 1:1. Minting access gated to authorized minters via OMS allowlist. WUSD circulates freely on Polygon once issued.

    New for Whish

    Anchorage Digital Issuance

    WUSD (Whish Dollar) is issued by Anchorage Digital Bank - federally chartered (OCC), with T-Bill backed reserves. Mint and burn operations route through Anchorage's regulated infrastructure.

    Live today

    Polygon Self-Custodial Wallets

    Whish uses Polygon's self-custodial wallet infrastructure for every end-user wallet - battle-tested in production with other partners, new deployment for Whish. AWS Nitro Enclave key management, social / email / passkey / OIDC sign-in, gas sponsored by Whish via Paymaster - Whish never holds user keys.
